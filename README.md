# Sombras de Manhattan

![Header](https://img.itch.zone/aW1nLzE4MDQwMjAwLnBuZw==/347x500/2%2Fp%2Fg.png)

**Sombras de Manhattan** es un juego de sigilo y puzzles ambientado en el Nueva York de los años 50. Investiga la corrupción de la mafia, resuelve acertijos y evita a "El Ojo" en una experiencia noir inmersiva.

## 🎯 Características

- 🕵️ **Gameplay de sigilo** con IA patrullando ("El Ojo")
- 🧩 **Puzzles desafiantes** basados en lógica y recolección de pistas
- 🎭 **Narrativa profunda** con personajes memorables
- 🎷 **Ambiente noir** con gráficos HDR y banda sonora de jazz
- 📜 **3 misiones principales** con múltiples objetivos
- 🗺️ **Sistema de mapa y minimapa** para navegación

## 🎮 Controles

| Acción | Tecla |
|--------|-------|
| Movimiento | `WASD` |
| Linterna | `F` |
| Ver misiones | `Q` |
| Mapa/Minimapa | `Shift` |
| Saltar | `Space` |
| Pausa | `Escape` |

## 🚀 Instalación

1. Descarga la última versión desde [itch.io](https://giod3v.itch.io/sombras-de-manhattan)
2. Descomprime el archivo ZIP
3. Ejecuta `SombrasDeManhattan.exe`
